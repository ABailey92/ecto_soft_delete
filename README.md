# EctoSoftDelete

Adds columns, fields, and queries for soft deletion with Ecto.

## Installation

Add to mix.exs:

```elixir
defp deps do
  [{:ecto_soft_delete, "0.1.0"}]
end
```

and do

```
mix deps.get
```

## Configuration

There are currently no configuration options.
